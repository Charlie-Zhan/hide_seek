# SKILL: Kenney 素材与地图资源管线

## 使用场景

当任务涉及 Kenney 素材选择、导入、图集、命名、地图搭建、道具、遮挡、碰撞或资源压缩时，阅读本文件。

---

## 1. 资源原则

- 仅使用已经确认可用于项目的素材包。
- 保留素材来源与授权说明，不删除 license 文件。
- 使用前统一整理命名，避免直接使用杂乱文件名。
- 不要把所有素材一次性导入首包。

---

## 2. MVP 推荐素材方向

- 角色：可爱小人、动物、食物角色或简单圆形角色。
- 地图：Top-down 风格 tiles、厨房、仓库、公园等。
- 道具：箱子、桶、盆栽、椅子、垃圾桶、篮子、小石头。
- UI：简洁按钮、面板、图标。
- 音效：轻快破碎、按钮、倒计时、胜利失败提示。

---

---

## 4. 图集建议

按用途拆图集：

```text
atlas_gameplay_props
atlas_characters
atlas_ui_common
atlas_map_kitchen_01
atlas_effects
```

不要把所有图都塞进一个巨大图集。微信小游戏首包和加载性能都要考虑。

---

## 5. 道具分类

每个道具配置至少包含：

```ts
interface PropConfig {
  id: string;
  spritePath: string;
  category: string;
  isDisguiseCandidate: boolean;
  isBreakable: boolean;
  isOccluder: boolean;
  collisionRadius?: number;
  visualRadius?: number;
}
```

分类：

| 类型 | 用途 |
|---|---|
| 可伪装可破坏 | 玩家可变成，找人者可打碎 |
| 可破坏不可伪装 | 用作诱饵和场景变化 |
| 遮挡物 | 草丛、柱子、树冠 |
| 大障碍物 | 墙、柜台、大桌子 |
| 纯装饰 | 地毯、花纹、墙饰 |

---

## 6. 地图搭建原则

地图不能太乱，必须有可记忆性。

推荐比例：

```text
成组小道具：40%
单独小道具：30%
遮挡物后道具：20%
诱饵道具：10%
```

每组小道具推荐 2~4 个，不要一组堆 8 个以上。

---

## 7. MVP 地图：小厨房

区域：

```text
左上：冰箱区
右上：灶台区
中央：餐桌区
左下：箱子堆
右下：垃圾桶和盆栽区
```

可伪装道具：

- 木箱；
- 垃圾桶；
- 盆栽；
- 椅子；
- 水桶；
- 食材篮。

遮挡物：

- 柱子；
- 半高柜台；
- 大桌子前景边缘；
- 植物或墙角装饰。

---

## 8. 导入验收

- [ ] 每个道具有统一 ID。
- [ ] 可伪装道具都有 Sprite。
- [ ] 可破坏道具都有破碎反馈资源或占位效果。
- [ ] 遮挡物位于前景层。
- [ ] 地图至少有 3 个视觉地标。
- [ ] 不同道具外形能被玩家快速识别。
- [ ] 资源来源和授权有记录。

---

## 9. 禁止事项

- 不要使用授权不清的素材。
- 不要把素材原始大包全部放进首包。
- 不要让地图杂物过多导致找人者完全无法记忆。
- 不要把玩家伪装道具做得明显比普通道具更亮或更大。
