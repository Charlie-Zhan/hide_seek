# Phase 00 Foundation

## Scope

This phase establishes the repository skeleton for the Cocos Creator client, Node.js server placeholder, and shared TypeScript contracts.

## Non-Goals

- No player movement.
- No round state machine.
- No WebSocket room implementation.
- No WeChat mini game build integration.
- No final art assets.

## Verification

Run:

```bash
npm run validate:phase00
npm run typecheck
```
