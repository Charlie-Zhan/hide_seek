# Generated Cat Player Sprites

Full-body cat player sprites generated for the MVP runtime.

- Generator: OpenAI image generation through Codex built-in `image_gen`
- Requested model path: gpt-image-2 style workflow
- Generated source image retained: `source/gpt_image_2_cat_sprite_sheet.png`
- Prompt record: `PROMPT.md`
- Generation date: 2026-05-15

Runtime variants are chroma-key cutouts from the generated sprite sheet and are
stored under `client/assets/resources/art/characters/cats/`.

These player cat sprites are not Kenney assets. Kenney assets in this project
currently cover only the selected prop and kitchen object sprites documented in
`client/assets/art/kenney/`.
