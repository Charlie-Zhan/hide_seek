# Generated Cat Back Attack Sprites

Source sheet for back-facing cat paw-swipe attack frames generated with the built-in image generation tool on 2026-05-15.

Runtime frames are exported to `client/assets/resources/art/characters/cat_animations/` with names such as `cat_orange_tabby_back_attack_1.png` and `cat_orange_tabby_back_attack_2.png`.

This sheet supplements `client/assets/art/generated/cat_directional_attacks/` so every facing bucket has two attack frames.
