# Prompt

Use case: stylized-concept

Asset type: 2D game sprite sheet for a Cocos Creator / WeChat mini-game, direction-specific cat walk frames

Primary request: Create a clean sprite sheet of cute full-body cats with front-facing and back-facing walk animations, matching polished 2D mobile game art.

Subject: five different cat skins in rows: orange tabby, gray tuxedo, calico, black cat, siamese. Six columns per row: front_idle, front_walk_1, front_walk_2, back_idle, back_walk_1, back_walk_2.

Style/medium: commercial-quality cute 2D hand-painted game sprites, crisp outlines, soft cel shading, full body, readable at small size.

Composition/framing: exact grid, 5 rows x 6 columns, each cell centered with generous padding, consistent character size and proportions. Front frames face toward the viewer/camera for moving down. Back frames show the cat's back, tail, ears, and rear legs for moving up. No side-view rotation; these must be true front/back poses.

Scene/backdrop: perfectly flat solid #00ff00 chroma-key background for background removal.

Constraints: no shadows, no ground plane, no text labels, no watermark, no props, no UI. Background must be one uniform #00ff00 color and #00ff00 must not appear on the cats. Keep limbs and tails separated enough for clean alpha extraction.
