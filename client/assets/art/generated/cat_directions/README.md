# Generated Cat Direction Sprites

Source sheet for front-facing and back-facing cat walk frames generated with the built-in image generation tool on 2026-05-15.

Runtime frames are exported to `client/assets/resources/art/characters/cat_animations/` with names such as `cat_orange_tabby_front_walk_1.png` and `cat_orange_tabby_back_walk_1.png`.

These are project-generated assets, not Kenney assets.
