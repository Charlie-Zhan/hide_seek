# Generated Cat Directional Attack Sprites

Source sheet for side/front/diagonal cat paw-swipe attack frames generated with the built-in image generation tool on 2026-05-15.

Runtime frames are exported to `client/assets/resources/art/characters/cat_animations/` with names such as `cat_orange_tabby_side_attack_1.png`, `cat_orange_tabby_front_attack_2.png`, and `cat_orange_tabby_diag_front_attack_1.png`.

The generated sheet contained nine usable separated sprites per cat row, so back-facing attack frames are recorded in `client/assets/art/generated/cat_back_attacks/` and merged into the runtime directional attack manifest.
