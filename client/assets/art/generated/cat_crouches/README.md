# Generated Cat Sitting Idle Sprites

Source sheet for directional sitting idle cat frames generated with the built-in image generation tool on 2026-05-15.

Runtime frames are exported to `client/assets/resources/art/characters/cat_animations/` with names such as `cat_orange_tabby_side_crouch.png`, `cat_orange_tabby_front_crouch.png`, and `cat_orange_tabby_diag_back_crouch.png`.

The runtime frame names keep the existing `*_crouch` suffix for compatibility with the animation state machine, but the art is a hind-legs-down sitting pose rather than a prone pose.

The right-facing side and diagonal frames are mirrored in code for left-facing movement.
