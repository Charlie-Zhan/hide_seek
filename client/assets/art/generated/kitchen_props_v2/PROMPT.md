# Prompt

Use case: stylized-concept

Asset type: 2D mobile game prop and furniture sprite sheet for a top-down hide-and-seek game

Primary request: Create a polished production-quality sprite sheet of cozy kitchen props and furniture for a 2D top-down / slightly three-quarter overhead mobile game. These assets should look like finished commercial game art rather than flat stickers.

Subject: one horizontal row of nine separate objects: wooden crate, blue trash bin, potted leafy plant, wooden dining chair, blue water bucket, woven food basket, stove counter, sink counter, kitchen counter island. Each object should be readable from overhead and match a cute cozy kitchen theme.

Style/medium: polished 2D cartoon game sprite art, soft cel shading, subtle painted texture, crisp edges, consistent perspective and scale, high quality mobile game asset style.

Composition/framing: exact 9 evenly spaced square cells in one row, each object centered with generous padding, no grid lines, no labels, full object visible.

Scene/backdrop: perfectly flat solid #ff00ff chroma-key background for background removal.

Constraints: no characters, no text, no watermark, no cast shadows, no floor plane, no gradients, no texture in the background. Do not use #ff00ff anywhere in the props.
