# Prompt

Use case: stylized-concept

Asset type: 2D game sprite sheet for a Cocos Creator / WeChat mini-game, diagonal cat walk frames

Primary request: Create a clean sprite sheet of cute full-body cats with true diagonal 3/4 walk animations for top-down movement, matching polished 2D mobile game art.

Subject: five different cat skins in rows: orange tabby, gray tuxedo, calico, black cat, siamese. Six columns per row: diag_front_right_idle, diag_front_right_walk_1, diag_front_right_walk_2, diag_back_right_idle, diag_back_right_walk_1, diag_back_right_walk_2.

Style/medium: commercial-quality cute 2D hand-painted game sprites, crisp outlines, soft cel shading, full body, readable at small size.

Composition/framing: exact grid, 5 rows x 6 columns, each cell centered with generous padding, consistent character size and proportions. Front-right diagonal frames are 3/4 front view moving down-right, with face and chest visible. Back-right diagonal frames are 3/4 back view moving up-right, with back, ears, tail, and rear legs visible. Do not create pure side-view or pure vertical-view poses.

Scene/backdrop: perfectly flat solid #00ff00 chroma-key background for background removal.

Constraints: no shadows, no ground plane, no text labels, no watermark, no props, no UI. Background must be one uniform #00ff00 color and #00ff00 must not appear on the cats. Keep limbs and tails separated enough for clean alpha extraction.
