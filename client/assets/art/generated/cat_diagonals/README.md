# Generated Cat Diagonal Sprites

Source sheet for diagonal cat walk frames generated with the built-in image generation tool on 2026-05-15.

Runtime frames are exported to `client/assets/resources/art/characters/cat_animations/` with names such as `cat_orange_tabby_diag_front_walk_1.png` and `cat_orange_tabby_diag_back_walk_1.png`.

The right-facing diagonal frames are mirrored in code for left-facing diagonal movement.

These are project-generated assets, not Kenney assets.
