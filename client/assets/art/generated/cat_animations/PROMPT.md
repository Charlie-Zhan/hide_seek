# Prompt

Use case: stylized-concept

Asset type: 2D mobile game animation sprite sheet for a Cocos Creator / WeChat mini game

Primary request: Create a clean production-quality sprite sheet of five cute full-body cat player characters with animation frames. The cats must be clearly walking-capable full-body cats, not heads or icons.

Subject: Five cat variants, one per row: orange tabby, gray tuxedo, calico, black cat, siamese. Each row shows the same cat across seven animation columns: idle standing, walk frame A, walk frame B, paw swipe attack windup, paw swipe attack follow-through, revealed surprised upright pose, dizzy knocked-down pose with tiny spiral stars near the head.

Style/medium: polished 2D cartoon game sprite art, cozy commercial mobile game quality, top-down / slightly three-quarter overhead view, crisp edges, soft cel shading, readable at small size, consistent proportions and lighting across every frame.

Composition/framing: exact 7 columns by 5 rows grid, evenly spaced square cells, each character centered with generous padding, consistent scale, full body visible including four paws and tail. No grid lines, no labels.

Scene/backdrop: perfectly flat solid #00ff00 chroma-key background for background removal.

Constraints: no clothing, no hats, no props except tiny dizzy stars only in the dizzy frame, no text, no watermark, no cast shadows, no floor plane, no gradients, no texture in the background. Do not use #00ff00 anywhere in the cats or stars.
